Table

A table is a database object that stores a collection of related data within a database. A table is the base element in the relational database. A table consist of columns and rows. The columns are fixed but the rows can change.The above sql code creates a table with columns id and name. Which have a datatype of int and string respectively.

Implementation

create table books(id int, name varchar(255))

Index

A database index is a data structure that is used to increase performance of a database's data retrival.

Types of indexes

Unique Key

A unique key is a constraint on a database column(s) that doesn't allow two rows with the same value on that column(s). The above two commands both create a unique key on the column name. The first one adds it when creating. The second one alters the table to add it.

Implementation

create table books(id int, name varchar(255) unique(name))

alter table books add unique(name)

Primary Key

A primary key is a constraint on a database column. It is the same as unique key except that there can't be more than one primary key in a table and it can't be null.

Implementation

create table books (id int primary key, name varchar(255) unique(name))

alter table books add primary key(id)

View

A database view is the result set of a stored query on the data. A database view can be viewed as a subset of the table the query was run in. The select statment creates a read only view. The update statment creates a read write view.

Implementation

select name from books where id=1

update books set name="The history of philosophy" from books where id=1

Cursor

A database cursor is a pointer to a certain row on a view. It has the advantage for large result sets(views). First it declares a cursor and sets its query. Second it declares a varchar variable then it opens the cursor it then fetches the data in the cursor to the name variable.

Implementation

declare name_cursor Cursor for select name from books where id=1

declare @name varchar(255)

open name_cursor

fetch name_cursor into @name

close name_cursor
